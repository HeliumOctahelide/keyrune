Keyrune v4 Demo

Dark Light

Complete Magic: the Gathering set symbol pictographic font. Each symbol in v4 has been redrawn and normalized with 3 glyphs for the base rarity symbol, the border, and an optional fixed white layer that some symbols have. Use the demo tool here to see a detailed view by set, or view the full set at once.

Symbol layers

Each set symbol has 2 layers, the base symbol and a border, with an optional 3rd layer when there's a fixed white portion.

Border styles

Default and common rarity borders are white, and other rarities are black. Built-in classes let you override these as well, which you can see in the rarities below.

Rarities

Common, uncommon, rare, mythic, and time-shifted rarity classes add color to the symbol glyph. Gradients are an optional class to more closely resemble the printed version in modern sets.

Many of the very early set symbols had no border or always used a white border. While Keyrune applies the modern configuration to all sets, it supports all border and color permutations.
